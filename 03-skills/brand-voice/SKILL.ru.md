# Навык голоса бренда

## Обзор
Этот навык обеспечивает сохранение согласованного голоса бренда, тона и сообщений во всех коммуникациях.

## Идентичность бренда

### Миссия
Помочь командам автоматизировать рабочие процессы разработки с помощью AI

### Ценности
- **Простота**: Делать сложное простым
- **Надёжность**: Железное выполнение
- **Расширение возможностей**: Раскрыть человеческое творчество

### Тон голоса
- **Дружелюбный, но профессиональный** — доступный без излишней неформальности
- **Чёткий и краткий** — избегать жаргона, объяснять технические концепции просто
- **Уверенный** — мы знаем, что делаем
- **Эмпатичный** — понимать потребности и болевые точки пользователей

## Руководство по написанию

### Делать ✅
- Использовать «вы» при обращении к читателям
- Использовать активный залог: «Claude генерирует отчёты», а не «Отчёты генерируются Claude»
- Начинать с ценностного предложения
- Использовать конкретные примеры
- Ограничивать предложения 20 словами
- Использовать списки для ясности
- Включать призывы к действию

### Не делать ❌
- Не использовать корпоративный жаргон
- Не покровительствовать и не упрощать чрезмерно
- Не использовать «мы считаем» или «мы думаем»
- Не использовать ЗАГЛАВНЫЕ БУКВЫ кроме выделения
- Не создавать стены текста
- Не предполагать технические знания

## Словарь

### ✅ Предпочтительные термины
- Claude (не «Claude AI»)
- Генерация кода (не «авто-кодирование»)
- Агент (не «бот»)
- Оптимизировать (не «революционизировать»)
- Интегрировать (не «синергизировать»)

### ❌ Избегать термины
- «Передовой» (переизбыток)
- «Изменяющий правила игры» (расплывчато)
- «Использовать» вместо «применять» (канцелярит)
- «Задействовать» (используйте «использовать»)
- «Смена парадигмы» (неясно)

## Примеры

### ✅ Хороший пример
«Claude автоматизирует процесс ревью кода. Вместо ручной проверки каждого PR, Claude проверяет безопасность, производительность и качество — экономя команде часы каждую неделю.»

Почему это работает: Чёткая ценность, конкретные преимущества, ориентация на действие

### ❌ Плохой пример
«Claude использует передовой AI для предоставления комплексных решений разработки программного обеспечения.»

Почему не работает: Расплывчато, корпоративный жаргон, нет конкретной ценности
