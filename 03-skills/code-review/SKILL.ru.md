# Навык ревью кода

Этот навык обеспечивает комплексные возможности ревью кода с фокусом на:

1. **Анализ безопасности**
   - Проблемы аутентификации/авторизации
   - Риски утечки данных
   - Уязвимости инъекций
   - Криптографические слабости
   - Логирование конфиденциальных данных

2. **Ревью производительности**
   - Эффективность алгоритмов (анализ Big O)
   - Оптимизация памяти
   - Оптимизация запросов к БД
   - Возможности кэширования
   - Проблемы параллелизма

3. **Качество кода**
   - Принципы SOLID
   - Паттерны проектирования
   - Соглашения именования
   - Документация
   - Покрытие тестами

4. **Поддерживаемость**
   - Читаемость кода
   - Размер функций (должен быть < 50 строк)
   - Цикломатическая сложность
   - Управление зависимостями
   - Типобезопасность

## Шаблон ревью

Для каждого проверяемого фрагмента кода предоставить:

### Сводка
- Общая оценка качества (1-5)
- Количество ключевых выводов
- Рекомендуемые приоритетные области

### Критические проблемы (если есть)
- **Проблема**: Чёткое описание
- **Расположение**: Файл и номер строки
- **Влияние**: Почему это важно
- **Серьёзность**: Критическая/Высокая/Средняя
- **Исправление**: Пример кода

### Выводы по категориям

#### Безопасность (если есть проблемы)
Список уязвимостей безопасности с примерами

#### Производительность (если есть проблемы)
Список проблем производительности с анализом сложности

#### Качество (если есть проблемы)
Список проблем качества кода с предложениями по рефакторингу

#### Поддерживаемость (если есть проблемы)
Список проблем поддерживаемости с улучшениями
