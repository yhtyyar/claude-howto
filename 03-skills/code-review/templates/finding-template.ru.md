### Пример кода

#### Текущий (проблемный)

```typescript
// Демонстрирует проблему N+1 запросов
const users = fetchUsers();
users.forEach(user => {
  const posts = fetchUserPosts(user.id); // Запрос на каждого пользователя!
  renderUserPosts(posts);
});
```

#### Предлагаемое исправление

```typescript
// Оптимизировано с JOIN-запросом
const usersWithPosts = fetchUsersWithPosts();
usersWithPosts.forEach(({ user, posts }) => {
  renderUserPosts(posts);
});
```
