# Оптимизация кода

Проверьте предоставленный код на следующие проблемы в порядке приоритета:

1. **Узкие места производительности** — выявить O(n²) операции, неэффективные циклы
2. **Утечки памяти** — найти неосвобождённые ресурсы, циклические ссылки
3. **Улучшение алгоритмов** — предложить более эффективные алгоритмы или структуры данных
4. **Возможности кэширования** — определить повторяющиеся вычисления
5. **Проблемы параллелизма** — найти гонки данных или проблемы с потоками

Отформатируйте ответ с:
- Серьёзность проблемы (Критическая/Высокая/Средняя/Низкая)
- Местоположение в коде
- Объяснение
- Рекомендуемое исправление с примером кода
