# Генератор API-документации

Сгенерируйте API-документацию:

1. Сканировать все файлы в `/src/api/`
2. Извлечь сигнатуры функций и JSDoc-комментарии
3. Организовать по эндпоинту/модулю
4. Создать markdown с примерами
5. Включить схемы запросов/ответов
6. Добавить документацию ошибок

Формат вывода:
- Markdown-файл в `/docs/api.md`
- Включить curl-примеры для всех эндпоинтов
- Добавить TypeScript-типы
