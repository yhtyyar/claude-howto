# Проверка покрытия тестами

Проанализировать покрытие и качество тестов:

1. Проверить процент покрытия тестами
2. Определить непокрытые пути кода
3. Проверить качество тестов
4. Предложить недостающие тестовые случаи
5. Убедиться, что граничные случаи охвачены
