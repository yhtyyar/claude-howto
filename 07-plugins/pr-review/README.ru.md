# Плагин ревью PR

Полный рабочий процесс ревью PR с проверками безопасности, тестирования, документации, качества кода и анализа производительности.

## Возможности

- Комплексное ревью pull request
- Автоматизированный анализ безопасности
- Верификация покрытия тестами
- Оценка влияния на производительность
- Проверки качества кода
- Проверка документации

## Установка

```json
{
  "plugins": ["github:your-org/pr-review-plugin"]
}
```

## Включённые команды

- `/review-pr` — Запустить полное ревью PR
- `/check-security` — Целевой анализ безопасности
- `/check-tests` — Проверка покрытия тестами

## Включённые субагенты

- **security-reviewer** — Ревью с фокусом на безопасности
- **test-checker** — Анализ покрытия и качества тестов
- **performance-analyzer** — Оценка влияния на производительность

## Включённые MCP-серверы

- **github** — Доступ к PR и управление ими

## Включённые хуки

- **PrePRMerge** — Автоматические проверки безопасности и тестов перед слиянием

## Использование

```bash
# Запустить полное ревью PR
/review-pr

# Выполнить только проверку безопасности
/check-security

# Проверить покрытие тестами
/check-tests
```

## Требования

- GitHub CLI настроен
- Доступ к репозиторию PR
- Тестовая инфраструктура настроена

## Конфигурация

```json
{
  "pr-review": {
    "min_test_coverage": 80,
    "security_level": "high",
    "require_docs_update": true
  }
}
```

## Пример рабочего процесса

1. Открыть PR в GitHub
2. `/review-pr` — полное ревью запущено параллельно
3. **security-reviewer** сканирует на уязвимости
4. **test-checker** оценивает покрытие тестами
5. **performance-analyzer** выявляет проблемы производительности
6. Консолидированный отчёт с выводами и предложениями
7. Необходимые изменения реализованы до слияния
