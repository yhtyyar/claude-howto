# Функция: `functionName`

## Описание
Краткое описание того, что делает функция.

## Сигнатура
```typescript
function functionName(param1: Type1, param2: Type2): ReturnType
```

## Параметры

| Параметр | Тип | Обязательный | Описание |
|---------|-----|-------------|---------|
| param1 | Type1 | Да | Описание param1 |
| param2 | Type2 | Нет | Описание param2 |

## Возвращаемое значение
**Тип**: `ReturnType`

Описание того, что возвращается.

## Исключения
- `Error`: При передаче некорректных входных данных
- `TypeError`: При передаче неправильного типа

## Примеры

### Базовое использование
```typescript
const result = functionName('value1', 'value2');
console.log(result);
```

### Расширенное использование
```typescript
const result = functionName(
  complexParam1,
  { option: true }
);
```
