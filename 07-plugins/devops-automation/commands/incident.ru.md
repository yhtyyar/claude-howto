# Реагирование на инциденты

Структурированный рабочий процесс реагирования на инциденты:

1. Создать запись об инциденте
2. Оценить серьёзность и влияние
3. Уведомить дежурную команду
4. Собрать диагностическую информацию
5. Координировать усилия по реагированию
6. Задокументировать решение
7. Запланировать post-mortem
