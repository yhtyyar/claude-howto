# Деплой приложения

Выполнить рабочий процесс деплоя:

1. Запустить предварительные проверки деплоя
2. Собрать приложение
3. Запустить тесты
4. Задеплоить в целевую среду
5. Выполнить проверки работоспособности
6. Уведомить команду в Slack
